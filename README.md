# Automobile workshop mangement

A web application for streamlined automobile workshop mangement.

## Features

- Secure authentication using **jwt**.
- Online bookings.
- Comprehensive admin pannel.
- Automated whatsapp messages.

## Tech-stack

- **ReactJS**
- **TailwindCSS**
- **NodeJS**
- **ExpressJS**
- **MangoDB**
